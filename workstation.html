My Workstation #

This page is a brief summary of my workstation and the software I use daily.

My Current Desktop #

My workstation is running Omarchy (Arch Linux) with the Hyprland desktop environment.

I highly recommend Omarchy for developers looking to try out Linux.

Workstation Specs
Mobo: ASUS PRIME X370-A
CPU: AMD Ryzen 5 1600X (6 cores @ 3.6GHz)
GPU: AMD Radeon RX 580 Series
Memory: Corsair 32GB DDR4 3000MHz

zed editor #

Zed is a fast, open-source, and collaborative code editor built in Rust, designed for high performance and features like real-time multiplayer collaboration, native Git integration, and "agentic" AI assistance. It uses GPU acceleration and other optimizations to ensure a smooth and responsive experience, even with large files. Key features include native debugging, edit predictions, a built-in terminal, and the ability to collaborate with AI and other developers in real time.

I use Zed daily and highly recommend it over other IDEs like VSCode.

get zed editor

lite-xl #

A lightweight, simple, fast, feature-filled, and extremely extensible text editor written in C, and Lua, adapted from lite.

Great alternative to editors like Sublime, or for workstations where Zed won't run.

get lite-xl

barrier #

Open-source KVM software. Eliminate the barrier between your machines.

ckb-next #

ckb-next is an open-source driver for Corsair keyboards and mice. It aims to bring the features of Corsair's proprietary CUE software to Linux operating systems.

get ckb-next

#

syncthing #

Syncthing is a continuous file synchronization program. It synchronizes files between two or more computers in real time, safely protected from prying eyes.

get syncthing

#

streamlink #

Streamlink is a CLI utility which pipes video streams from various services (twitch, youtube, etc) into a video player such as VLC.

get streamlink
