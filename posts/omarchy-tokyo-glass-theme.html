omarchy tokyo glass theme #

October 23rd, 2025 #

So I've recently switched to Omarchy on my workstation and I really enjoy the Tokyo Night theme. I also really enjoy glass themes and rounded borders so I went ahead and modified the theme to add a bit of glass.

features
- tokyo night theme
- glass waybar
- rounded glass borders
- glass hyprlock input

I decided to package the theme and share it with everyone via GitHub.

Here is how to install it via your terminal:

omarchy-theme-install https://github.com/coreydemarse/omarchy-tokyo-glass-theme

enable border-plus-plus plugin for better borders

hyprpm add https://github.com/hyprwm/hyprland-plugins
hyprpm update
hyprpm enable borders-plus-plus
