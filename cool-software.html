barrier #

Open-source KVM software. Eliminate the barrier between your machines.

get barrier

#

obsidian #

Obsidian is a powerful knowledge base on top of a local folder of plain text Markdown files.

get obsidian

#

insomnia #

The open-source, cross-platform API client for GraphQL, REST, and gRPC.
Build APIs that work.

get insomnia

#

lite-xl #

A lightweight, simple, fast, feature-filled, and extremely extensible text editor written in C, and Lua, adapted from lite.

get lite-xl

#

beekeeper studio #

Modern and easy to use SQL client for MySQL, Postgres, SQLite, SQL Server, and more. Linux, MacOS, and Windows.

get beekeeper studio

#

ckb-next #

ckb-next is an open-source driver for Corsair keyboards and mice. It aims to bring the features of Corsair's proprietary CUE software to Linux operating systems.

get ckb-next

#

syncthing #

Syncthing is a continuous file synchronization program. It synchronizes files between two or more computers in real time, safely protected from prying eyes.

get syncthing

#

neofetch #

Neofetch displays information about your operating system, software and hardware in an aesthetic and visually pleasing way. Written in bash 3.2+

get neofetch

#

streamlink #

Streamlink is a CLI utility which pipes video streams from various services (twitch, youtube, etc) into a video player such as VLC.

get streamlink
